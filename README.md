# Junkie

Junkie is a simple ruby gem for cloning git repos from jenkins views

## Installation

Add this line to your application's Gemfile:

    gem 'junkie'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install junkie

## Usage
install:

    $ gem install junkie

then run using:

    $ junkie
